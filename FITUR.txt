OVERVIEW:
=========
Sistem Lelang Online adalah aplikasi web yang memungkinkan pengguna
untuk melakukan lelang barang secara online dengan 3 level user:
Administrator, Petugas, dan Masyarakat.


FITUR BERDASARKAN USER LEVEL:
==============================

A. ADMINISTRATOR
================
✓ 1. LOGIN & LOGOUT
     - Login dengan username & password
     - Session management
     - Auto redirect setelah login
     - Logout dengan destroy session

✓ 2. DASHBOARD
     - Total User (statistik)
     - Total Lelang (statistik)
     - Total Barang (statistik)
     - Total Transaksi Selesai (statistik)
     - Tabel Lelang Terbaru
     - Grafik visual (cards dengan icon)

✓ 3. MANAJEMEN DATA BARANG
     - Tambah barang baru
     - Edit data barang
     - Hapus barang

     - Input deskripsi barang
     - Set status barang (pending/dibuka/ditutup)
     - Tabel data barang dengan search

✓ 4. MANAJEMEN TOTAL LELANG
     - Lihat semua lelang
     - Detail lelang
     - Monitor status lelang
     - Lihat pemenang lelang
     - Tracking harga lelang

✓ 5. MANAJEMEN PEMBAYARAN
     - Lihat daftar pembayaran

✓ 6. GENERATE LAPORAN
